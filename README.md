#  Gym Website

A modern, responsive **single-page gym website** built with **HTML5**, **CSS3**, and **JavaScript**.  
The website showcases gym **facilities, services, trainers, schedules, membership plans, testimonials, FAQs**, and a **contact form**, all in a smooth scrollable layout enhanced with animations.



[Live Demo](https://saeedasaad.github.io/HexSoftwares_Project_gym_website/)


---

##  Features

###  Core Features
- Fully responsive layout (mobile, tablet, desktop)
- Smooth scroll navigation with active section highlighting
- Elegant UI design with modern color palette
- Sticky navigation bar

###  Animations & Interactivity
- GSAP-powered entrance and scroll animations
- Hero section reveal effects
- Scroll-triggered animations on all sections
- Loading screen animation
- Back-to-top floating button
- Animated FAQ accordion
- Testimonials slider with next/prev navigation

###  Website Sections
- Hero section  
- Facilities section  
- Services & class categories  
- Trainers section  
- Class schedule with dynamic filtering  
- Membership pricing plans  
- Testimonials slider  
- FAQs  
- Contact form with live character counter
