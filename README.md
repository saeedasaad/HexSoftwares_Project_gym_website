#  Gym Website

A modern, responsive single-page gym website built with **HTML5**, **CSS3**, and **JavaScript**. This project showcases gym facilities, services, trainers, schedules, membership plans, testimonials, FAQs, and a contact form — all in a smooth scrollable layout with interactive animations.

##  Live Demo

 [View Live Demo](https://saeedasaad.github.io/HexSoftwares_Project_gym_website/)  

---

##  Features

-  Responsive navigation menu with scroll-based active highlighting
-  Hero section with GSAP-powered entrance animations
-  Facilities & services cards with scroll-reveal effects
-  Trainer profiles with animated appearance
-  Class schedule with filterable categories
-  Membership plans with pricing cards
-  Testimonials slider with navigation and indicators
-  FAQ accordion for common questions
-  Contact form with character count and smooth scroll-to-top button
-  Loading screen animation
-  GSAP ScrollTrigger animations across all sections

---

##  Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Responsive layout, animations, and styling
- **JavaScript (ES6+)** – DOM manipulation, interactivity
- **GSAP** – Advanced animations and scroll effects
